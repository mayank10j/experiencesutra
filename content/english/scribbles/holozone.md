</div><div style="clear:both"></div><div style="padding-bottom:4px;"></div><p>In an&nbsp;era where head-mounted virtual retinal displays such as the <a class="jive-link-external-small" href="https://www.microsoft.com/microsoft-hololens/en-us" rel="nofollow">Hololens</a> and <a class="jive-link-external-small" href="http://www.magicleap.com/#/home" rel="nofollow">Magic Leap</a> are taking augmented reality to greater heights, here’s a rather retro and cost efficient approach to interacting with holograms in the real world.</p>
<p>Holozone&nbsp;uses <a class="jive-link-external-small" href="https://en.wikipedia.org/wiki/Pepper%27s_ghost" rel="nofollow">Pepper’s ghost</a> in order to augment virtual objects in a physical space and a variety of mechanisms in order to interact with them.</p>
<p>An installation of this sort could be used in two fundamental ways:</p>
<p>1. To overlay real objects with virtual components<span style="font-weight: 400;"> such that</span>&nbsp;the User Interface reacts to the object, with&nbsp;emphasis on the real object and the interface working around it.</p>
<p><a href="http://experiencesutra.com/wp-content/uploads/2015/12/ezgif.com-optimize1.gif"><img alt="ezgif.com-optimize" class="aligncenter wp-image-397 size-full" height="320" src="http://experiencesutra.com/wp-content/uploads/2015/12/ezgif.com-optimize1.gif" width="480"/></a></p>
<p>2. To interact with virtual entities using objects or gestures in the physical world.</p>
<p><a href="http://experiencesutra.com/wp-content/uploads/2015/12/IMG_20151130_173955035.jpg"><img alt="IMG_20151130_173955035" class="aligncenter wp-image-385 size-large" height="506" src="http://experiencesutra.com/wp-content/uploads/2015/12/IMG_20151130_173955035-1024x576.jpg" width="900"/></a></p>
<p>Cameras could be used to identify objects or detect gestures.</p>
<p>In the sample below we have used the <a class="jive-link-external-small" href="https://www.leapmotion.com/" rel="nofollow">Leap Motion</a> to track a user’s hands.</p>
