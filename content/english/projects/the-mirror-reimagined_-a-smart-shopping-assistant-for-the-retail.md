</div><div style="clear:both"></div><div style="padding-bottom:4px;"></div><p>We started experimenting with the idea of a smart shopping assistant for the physical apparel stores around this time <a href="http://experiencesutra.com/projects/the-interactive-mirror/" target="_blank">last year</a>. We had envisioned the Mirror to be a personalized shopping assistant with the capabilities of a recommendation system. Recommendation systems aren’t anything that is new. In fact, smart recommendations have long transformed the world of e-business. Netflix’s recommendation engine generates <a href="https://www.businessinsider.in/Why-Netflix-thinks-its-personalized-recommendation-engine-is-worth-1-billion-per-year/articleshow/52754724.cms" target="_blank">$1 billion yearly revenue</a>. In 2012, Amazon reported 29% sales increase during their second fiscal quarter. <a href="http://fortune.com/2012/07/30/amazons-recommendation-secret/" target="_blank">The success story of Amazon</a> was accredited to the recommendation system they had in place. Fashion e-commerce websites offer smart recommendations based on customer preference. Shopping history, wishlist items etc. indicate the customer’s likings. You’d add something to your shopping cart or to your wish list, and the mobile e-commerce app or the website would give you a list of recommendations. Your wishlist shows your intent.</p>
<p>Inventory visibility directly connects to the number of sales. And this is the reason recommendation systems are so successful. The more you show, the more you sell – is the mantra. But how can you have such a recommendation system for a physical setup? It has to serve the same purpose as in an e-commerce website and has to be seamless, as well. So, we envisioned a user journey. When a customer picks up some clothing item and tries to see how they would look on it in front of a mirror – by casually holding the cloth in front of them, the mirror would surprise them by showing more similar options they could purchase.</p>
<p><a href="http://experiencesutra.com/wp-content/uploads/2018/05/Mirror_StoryBoard_02MAY181.jpg"><img alt="Mirror StoryBoard" class="aligncenter size-full wp-image-794" height="1080" src="http://experiencesutra.com/wp-content/uploads/2018/05/Mirror_StoryBoard_02MAY181.jpg" width="2430"/></a></p>
<p>When designing the Mirror, we had the following pointers in mind.</p>
<ol>
<li>The experience has to be absolutely frictionless for shoppers.</li>
<li>The Mirror must improve the sales figures by bringing forward the inventory.</li>
</ol>
<p>The working prototype of the Mirror that we have demonstrated very recently at the Great Indian Developer Summit (Asia’s largest developer summit, at Indian Institute of Science, Bengaluru), 2018, serves our intention. It has the following functionalities.</p>
<ol>
<li>Classifies Men’s and Women’s categories, automatically.</li>
<li>Recognizes apparel type and shows a list of recommendations of similar looking clothes.</li>
<li>Suggests other items (complimentary/matching clothes) that the shopper might like to buy.</li>
<li>Presents deals/available sizes and an option to shortlist items the shopper would like to try-on inside a trial room.</li>
</ol>

<p>&nbsp;</p>
<p>So, how does the Mirror work?</p>
<p>The Mirror is powered by state-of-the-art Deep Learning. We have trained four different deep learning models for the following tasks.</p>
<ol>
<li>Person Detection.</li>
<li>Face Detection.</li>
<li>Gender Detection.</li>
<li>Apparel Category Classification.</li>
</ol>
<p>We have prepared a dataset of 30k+ images to train our DCNN model for apparel category classification. Detailed explanations of our deep learning architectures can be found in our <a href="http://experiencesutra.com/experiments/deep-learning-in-fashion/" target="_blank">previous blog</a>. Accuracies of our apparel classification model are shown in the following graph.</p>
<p><a href="http://experiencesutra.com/wp-content/uploads/2018/05/Screen-Shot-2018-05-02-at-1.21.15-PM.png"><img alt="accuracies" class="aligncenter size-full wp-image-778" height="768" src="http://experiencesutra.com/wp-content/uploads/2018/05/Screen-Shot-2018-05-02-at-1.21.15-PM.png" width="1788"/></a></p>
<p>After apparel category classification, in the final stage of generating a list of similar items, we rank the store inventory based on features extracted from the query image. Query features are extracted from intermediate DCNN layers. Deep Convolutional Neural Nets can be utilized to extract features from an image. Features are represented in a hierarchical manner in a CNN. First few layers of a CNN extract low-level features like color and gradient. More abstract features like edges, textures are extracted from deeper layers. We have prepared a database of features that map to items in the store inventory, beforehand. This database is well optimized (using state-of-the-art clustering algorithms) for storage and search. The inference architecture for recommendation generation using similarity search is depicted in the diagram below.</p>
<p><a href="http://experiencesutra.com/wp-content/uploads/2018/05/mirror-arch.png"><img alt="mirror-arch" class="aligncenter size-full wp-image-780" height="1062" src="http://experiencesutra.com/wp-content/uploads/2018/05/mirror-arch.png" width="2288"/></a></p>
<p>The newest feature in our Mirror is cross-recommendations. Showing relevant items that shoppers can buy is a sales booster. We want our Mirror to display relevant items, such as trousers, handbags, backpacks etc. for a complete look recommendation. Currently, we are suggesting trousers with shirts, tops, t-shirts etc.</p>
<p>We have more feature extensions and improvements planned for the Mirror. Stay tuned on experiencesutra for updates.</p>
